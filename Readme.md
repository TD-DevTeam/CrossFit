# CrossFit Log Managment System

## Due date: 9/15

초기 목표:

* 자기 자신의 운동내역을 추적할 수 있어야 함.
    * 자기 자신의 운동을 기록할 수 있어야 함.
    	* 운동 종목은 관리자 한명이 정해야 함.
    	* 종목에 따라 기록단위가 정해짐.
    * 자기 자신의 운동 기록을 편하게 (한눈에) 볼 수 있어야 함.
* 자기 자신을 알아 볼 수 있어야 함.
	* 사용자 데이터베이스가 있어야 함.
	* 로그인 시스템이 있어야 함. 
* 관리자 시스템 필요
	* 운동 종목을 정해야 함.
		* 운동 종목이 디비에 들어가 있어야 함. 
	* 사용자 등록 상황을 체크 할 수 있음.
